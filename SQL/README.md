# SQL & Relational Database Scripts

This folder contains SQL scripts demonstrating relational database management, PL/SQL stored procedures, and transactional processing.

## 📄 Included Scripts
1. **`transactional_order_processing.sql`** — Demonstrates sequence usage, multi-table updates, database transactions (`COMMIT`, `ROLLBACK`), and cascade deletes across customer, order, and product schemas.
2. **`hr_employee_procedures.sql`** — Contains stored procedures designed to automate order ingestion, manage identity columns, and execute standard transactional blocks.
